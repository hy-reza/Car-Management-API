## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/hy-reza/Car-Management-API.git
   ```
2. Install packages
   ```sh
   yarn add
   ```
3. Next Step

   ```sh
   - setting config.js / .env
   - yarn db:create
   - yarn db:migrate
   - yarn db:seed:all
   ```

4. Run
   ```sh
   yarn start
   ```
